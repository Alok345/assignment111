## 🚀 Getting Started

### Prerequisites
Ensure you have [Node.js](https://nodejs.org/en/download/) installed on your machine. We recommend the latest LTS version.

### Installation
Follow these steps to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Alok345/assignment111.git
   cd assignment111
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```

Your application should now be running at `http://localhost:3000`.
